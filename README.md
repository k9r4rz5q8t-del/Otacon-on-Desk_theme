## Otacon on Desk Theme v1.0.0

A fan-made Otacon pixel theme for Clawd on Desk.

This release includes:
- Custom Clawd theme configuration (`theme.json`)
- GIF/SVG animation assets
- Idle eye-tracking SVG
- Notification / permission animations
- Optional sound overrides (`confirm.wav`, `complete.wav`)

## Installation

1. Download `otacon-theme.zip`.
2. Unzip it.
3. Copy `otacon-theme/` to:

   `~/Library/Application Support/clawd-on-desk/themes/otacon-theme/`

4. Optional: copy `sound-overrides/otacon-theme/` to:

   `~/Library/Application Support/clawd-on-desk/sound-overrides/otacon-theme/`

5. Restart Clawd on Desk and select the `Otacon` theme.
